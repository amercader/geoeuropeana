Europeana API geo-wrapper

This is demo application that uses the Europeana API geo wrapper to perform spatial queries. It was built during the first Europeana Hackathon.

The wrapper is very slow, so please be patient when querying.

For more details, please have a look at this post : http://amercader.net/blog/2011/04/europeana-hackathon

Controls:Drag to pan, wheel to zoom, Ctrl + drag to query
Draw a box on the map to start
